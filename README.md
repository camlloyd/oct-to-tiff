# oct-to-tiff

Convert optical coherence tomography files to OME-TIFF.

## Supported scan patterns

Due to limited test data, only the following scan patterns are currently supported:
- 3D Cornea
- 3D Disc
- 3D Retina
- Angle
- Cornea Cross Line
- Cross Line
- Enhanced HD Line
- GCC
- Grid
- HD Angio Retina
- Line
- Pachymetry Wide
- Radial Lines
- Raster

## Requirements

Requires Python 3.7 or higher.
