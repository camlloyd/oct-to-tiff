# oct-to-tiff

A command line tool for converting optical coherence tomography angiography (OCTA) data.

## Supported scan patterns

Due to limited test data, only the following scan patterns are currently supported:
- 3D Cornea
- 3D Disc
- 3D Retina
- 3D Widefield
- 3D Widefield MCT
- Angio Disc
- Angio Retina
- Angle
- Cornea Cross Line
- Cornea Line
- Cross Line
- Enhanced HD Line
- GCC
- Grid
- HD Angio Disc
- HD Angio Retina
- Line
- Pachymetry Wide
- Radial Lines
- Raster

## Requirements

Requires Python 3.7 or higher.
